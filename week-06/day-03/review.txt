To create a DB: 
- <create database *name of DB*;>
- Select the command and press the run button.

To create a table {ex.table: users for blog DB}:
- Select DB: <use *name of DB*;>

To create columns:
- <create table *name of table1* (
	id int -- 4 bytes, 32 bits{limitation of the largest number we can store} OR
	id bigint primary key{first column} auto_increment{auto generate}, -- 8 bytes, 64 bits
	username text OR varchar(upper limit size) unique{validation + performance}, -- constraint
	age int,
	email varchar(255)
);>

To add rows: 
- <insert into *name of table1* (*name of columns*)>
- <values (values of corresponding columns)>
- <select *name of columns* from *name of table*>

To delete a row:
- <delete from *name of table* where id = *id number*;>

Once a row is deleted, its id is not reused for new rows. 

To update a row:
- <update *name of table* set *name of column* = old value + static value>
	<where *name of column* = current value>
